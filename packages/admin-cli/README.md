# Installation

```
npm i -g @nmshd/connector-admin-cli
```

# Usage

## Identity commands

```
connector-admin identity [command]

Identity related commands

Commands:
  connector-admin identity init    initialize the identity
  connector-admin identity status  show the status of the identity

Options:
      --version  Show version number                                   [boolean]
  -h, --help     Show help                                             [boolean]
```

## Identity deletion commands

```
connector-admin identityDeletion [command]

Identity deletion related commands

Commands:
  connector-admin identityDeletion init     initialize the identity deletion
  connector-admin identityDeletion approve  approve the identity deletion
  connector-admin identityDeletion cancel   approve the identity deletion
  connector-admin identityDeletion reject   reject the identity deletion

Options:
      --version  Show version number                                   [boolean]
  -h, --help     Show help                                             [boolean]
```
